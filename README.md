## 📖 Overview

TradeAcademy is a modern, self-paced trading education platform designed to help beginners progress from market fundamentals to creating a complete trading plan.

The platform provides structured learning, realistic trading simulation, risk-management tools, calculators, glossaries, and trading templates—all running completely offline in a single HTML application.

## ✨ Features

### 📚 Learning System
- 56 structured lessons
- 8-week guided curriculum
- Interactive quizzes and knowledge checks
- Indian Equity, Crypto, and Forex modules
- Trading psychology and risk management

### 📊 Trading Simulator
- Virtual ₹1,00,000 portfolio
- Interactive market simulation

- Order book visualization

### 🧮 Trading Calculator
- Position sizing calculator
